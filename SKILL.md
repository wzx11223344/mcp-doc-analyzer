# SKILL.md - 文档分析MCP服务器技能文档

## 概述

mcp-doc-analyzer 是一个基于 MCP (Model Context Protocol) 协议的文档分析服务器，为 AI 客户端提供 21 个文档处理和分析工具。

## 核心能力

### 文档读取 (reader.py)

读取多种格式文件并返回结构化摘要：

- **纯文本文件**：支持 .txt, .md, .log 格式，自动检测编码（UTF-8/GBK/GB2312/Latin-1）
- **CSV 文件**：解析列信息、数据预览、非空值统计，支持自定义返回行数
- **JSON 文件**：解析数据结构、类型识别、递归结构分析、格式化预览
- **文件头/尾读取**：快速预览文件开头或结尾，适用于日志分析
- **目录列表**：支持 glob 模式匹配，返回文件/目录列表及大小信息

### 文本分析 (analyzer.py)

提供多维度文本分析：

- **词频统计**：jieba 中文分词 + Counter 统计，支持中英文混合文本
- **关键词提取**：TF-IDF 双引擎（jieba.analyse + sklearn TfidfVectorizer）
- **句子分析**：句子分割、长度统计、最长/最短句识别
- **可读性评分**：中文改编版 + 英文标准 Flesch-Kincaid，含年级水平评估
- **情感分析**：基于中英文情感词典的正面/负面/中性判断
- **语言检测**：Unicode 范围检测，支持中/英/日/韩/混合

### 摘要生成 (summarizer.py)

智能文本摘要和结构化：

- **抽取式摘要**：TextRank 算法（句子相似度矩阵 + PageRank 迭代），支持自定义摘要长度
- **文本统计**：字符/词/句/段落统计 + 阅读时间估算（中文350字/分钟，英文225词/分钟）
- **相似句子检测**：改进 Jaccard 相似度，双向重叠率加权，可配置阈值
- **自动大纲**：Markdown 标题解析 + TextRank 要点提取 + 关键词提取

### 数据统计 (statistics.py)

专业数据分析工具：

- **描述性统计**：15项指标（均值/中位数/标准差/方差/四分位数/偏度/峰度/变异系数等）
- **频率分布**：等宽分箱 + 频率/累计频率 + 可视化柱状图
- **相关性分析**：Pearson + Spearman + Kendall 三种相关系数 + 显著性检验
- **趋势分析**：线性回归（斜率/截距/R²/p值/标准误差）+ 移动平均
- **异常值检测**：IQR 法（1.5倍四分位距）和 Z-score 法（3倍标准差）

## 技术实现

### 算法

| 算法 | 用途 | 实现 |
|------|------|------|
| TextRank | 抽取式摘要 | PageRank 迭代 + 句子相似度矩阵 |
| TF-IDF | 关键词提取 | jieba.analyse + sklearn TfidfVectorizer |
| Jaccard | 句子相似度 | 改进版：Jaccard * 0.5 + 双向重叠率 * 0.5 |
| Flesch-Kincaid | 可读性 | 英文标准版 + 中文改编版 |
| Pearson/Spearman/Kendall | 相关性 | scipy.stats |
| 线性回归 | 趋势分析 | scipy.stats.linregress |
| IQR/Z-score | 异常值 | numpy + 手动实现 |

### 依赖库

| 库 | 版本要求 | 用途 |
|----|----------|------|
| mcp | >=1.0.0 | MCP 协议 + FastMCP 框架 |
| numpy | >=1.24.0 | 矩阵运算、数组操作 |
| scipy | >=1.10.0 | 统计分析（stats模块） |
| jieba | >=0.42.1 | 中文分词、关键词提取 |
| scikit-learn | >=1.3.0 | TF-IDF 向量化 |

## 使用场景

1. **文档处理流水线**：读取文件 -> 分析内容 -> 生成摘要 -> 统计数据
2. **日志分析**：读取日志文件尾部 -> 词频统计 -> 异常检测
3. **内容分析**：文本统计 -> 情感分析 -> 关键词提取 -> 可读性评分
4. **数据探索**：CSV读取 -> 描述性统计 -> 频率分布 -> 异常值检测
5. **学术写作**：句子分析 -> 相似句子检测 -> 自动大纲 -> 阅读时间估算

## 输出格式

所有工具返回 Markdown 格式字符串，包含：
- 标题分节（## 级别）
- 结构化表格（Markdown table 语法）
- 统计数值（保留适当小数位）
- 解读说明文本
- 数据预览

## 工具调用示例

```
# 读取文件
read_text_file(path="/path/to/document.md")

# 词频统计
word_frequency(text="文本内容...", top_n=20)

# 生成摘要
extractive_summary(text="长文本...", num_sentences=3)

# 描述性统计
describe_numeric(values=[1, 2, 3, 4, 5, 100])
```
